# Mochilista

## Objetivo

Aplicación web para preparar mochilas mediante listas de equipaje. El usuario puede utilizar listas predeterminadas (plantillas) y crear listas propias. En el plan Premium se añaden funciones avanzadas como clasificación por tipo de artículo, cálculo de peso y publicación de listas para reutilización por otros usuarios Premium.

---

## Funcionalidades

### 1) Usuarios y planes

- Registro e inicio de sesión.
- Dos tipos de usuario:
  - **Free**
  - **Premium** (el cambio a Premium se simula, sin pagos reales).

### 2) Listas predeterminadas (plantillas)

- El sistema incluye **5 listas predeterminadas** creadas por el desarrollador.
- Un usuario **Free** puede activar **solo 1** plantilla a la vez.
- En la plantilla activa:
  - Se permite **marcar/desmarcar checks** (estado propio por usuario).
  - Se permite **reiniciar** el checklist.
  - No se permite modificar el contenido (no añadir/editar/borrar ítems).

### 3) Lista personalizada (Free)
- Un usuario **Free** puede crear **1 lista personalizada**.
- Máximo **20 ítems**.
- Cada ítem incluye:
  - nombre (obligatorio)
  - cantidad (opcional)
  - check (opcional)
- En Free **no existe** el peso ni la clasificación por tipo.

### 4) Funcionalidades Premium en listas propias
- Un usuario **Premium** puede crear varias listas.
- Puede clasificar cada ítem con un **tipo de artículo** (catálogo fijo definido por el desarrollador; selección mediante desplegable).
- Puede introducir **peso** por artículo (guardado internamente en gramos).
- La app calcula:
  - **peso total** de la lista
  - **peso por tipo de artículo** si el usuario agrupa la vista por tipo
- El check **no afecta** al cálculo de peso.

### 5) Publicación y reutilización de listas (solo Premium)
- Un Premium puede marcar una lista como **pública**.
- Las listas públicas tienen **1 categoría** (catálogo fijo definido por el desarrollador):
  - General, Urbano, Naturaleza, Trekking, Camping, Campervan, Escalada, Pesca
- Un Premium puede:
  - **buscar/explorar** listas públicas por categoría
  - **descargar/clonar** una lista pública para personalizarla en su cuenta

### 6) Exportación a PDF (solo Premium)
- Un Premium puede exportar una lista a PDF con:
  - cabecera (nombre de lista + usuario creador)
  - tabla de artículos (nombre, cantidad y check)
