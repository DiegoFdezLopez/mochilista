# Tecnologías y arquitectura

## Frontend

- **React** (interfaz y enrutado).
- **Tailwind CSS** (estilos).
- Uso de **Google Stitch** como apoyo para prototipar y acelerar el diseño, integrando después las pantallas en React/Tailwind.

## Backend

- **Spring Boot (Java)** como API REST:
  - autenticación (registro/login)
  - lógica de negocio (límites Free/Premium, publicación, clonación, etc.)
  - exportación a PDF (si se implementa desde backend)

## Base de datos

- **MySQL** para persistencia:
  - usuarios
  - listas e ítems
  - catálogos (categorías y tipos de artículo)
  - plantillas predeterminadas
